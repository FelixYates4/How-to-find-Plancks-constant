How to Calculate Planck's Constant using Excel

Learn how to perform data analysis in Excel to find Planck's constant with formulas, graphs, and error bars.

1. Set up Your Data

Input LED wavelength (in nm) and threshold voltage (in V) into two seperate columns. convert the wavelength into metres by multiplying by 10-9.

2. Perform Calculations using Formulas

Use Excel formulas to calculate the frequency from the wavelength and the energy using the threshold voltage. For example, calculate the frequency with:

=3*10^8 / (A2*10^-9)

Then calculate energy with:

=1.602*10^-19 * Vth

3. Plot the Graph with a Line of Best Fit

To create a graph:

  1. Select your data (frequency(x) and energy(y)).
  2. Insert a scatter plot from the Excel toolbar.
  3. Right-click on the plot and add a trendline (line of best fit).
  4. Display the equation on the graph to find the slope.

4. Add Error Bars

Click on your graph, then click on the Chart Elements icon. Select Error Bars and customize them according to your data. Add error margins based on your measurements for wavelength and voltage.

5. Calculate Planck's Constant

The slope of the line of best fit represents Planck's constant.
